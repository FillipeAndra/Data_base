
## StartUML

- software para fazer uml do esquema do banco de dados (tem na AUR ou usa o debtap que também tem na AUR para converter o .deb)

## brmodelo 3.0

- software para fazer uml do esquema do banco de dados

## MYSQL, PostgreSQL, SQL Server, Oracle Express

- Servidores e SGBDs (sistemas de gerenciamento de bancos de dados) para Trabalhar com Bancos de dados com SQL (structured query language)

## Virtual box ou qemu 

- ambiente de virtualização para o diversos servidores (pretendo só usar para o sql server quando for rodar no arch)

## Vscode / DBEAVER / Notepad ++

- Ferramentas de scripts de múltipla linguagem que podem ser usadas para trabalhar com sql, à exceção do dbeaver que é ferramenta focada à conexão e manipulação de bancos de dados.
